## 🧠 苏格拉底式反问机制（核心特色）

### 3 大核心原则

1. **不满足于表面答案** → 追问"为什么"
2. **不假设方案完美** → 追问"真的更好吗"
3. **不忽略潜在风险** → 追问"如果失败怎么办"

### 反问模板（每次迭代必备）

```markdown
## 苏格拉底式反问（迭代 XX）

### 问题 1: "为什么需要这个改进？"
- 回答：[当前方案的理由]
- 追问：[哪些具体问题需要解决？]
- 再追问：[如何证明这些问题是真实存在的？]

### 问题 2: "改进后真的更好吗？"
- 回答：[理论上的改进效果]
- 追问：[如何验证改进效果？]
- 再追问：[验证标准是什么？]

### 问题 3: "如果失败了怎么办？"
- 回答：[回滚方案]
- 追问：[回滚成本多大？]
- 再追问：[如何降低失败风险？]
```

## 📊 数据分析规范

### 数据预处理

```python
# 1. 读取数据
import pandas as pd
df = pd.read_csv('data.csv')

# 2. 检查数据质量
print(f"行数：{len(df)}")
print(f"缺失值：{df.isnull().sum()}")
print(f"数据类型：{df.dtypes}")

# 3. 基本统计
print(df.describe())
```

### 分析维度

1. **描述性统计** - 均值、中位数、标准差
2. **分布分析** - 直方图、箱线图
3. **相关性分析** - 相关系数矩阵
4. **异常检测** - Z-score、IQR 方法

### 报告格式

```markdown
📊 数据分析报告

## 数据概览
- **文件**: data.csv
- **行数**: 1000
- **列数**: 15
- **缺失值**: 5%

## 关键发现
- ✅ 发现 1: XXX 趋势
- ✅ 发现 2: XXX 分布
- ⚡ 发现 3: XXX 异常

## 建议
1. 建议 1
2. 建议 2
```
