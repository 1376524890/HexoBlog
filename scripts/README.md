# 铅笔小说爬虫脚本使用说明

## 功能特点

- ✅ 根据小说 URL 下载整本小说
- ✅ 自动获取所有章节
- ✅ 合并所有章节为 TXT 文件
- ✅ 处理反爬机制（User-Agent、请求间隔）
- ✅ 保存到指定目录
- ✅ 使用 tldextract 确保同域访问

## 依赖安装

```bash
# 创建虚拟环境（如果还没有）
python3 -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r scripts/requirements.txt
```

## 使用方法

### 基本用法

```bash
# 使用虚拟环境运行
cd /home/claw/.openclaw/workspace
source venv/bin/activate

# 下载小说
python scripts/novel_crawler.py -u "http://www.5963.net/book/19720/"
```

### 指定输出目录

```bash
python scripts/novel_crawler.py -u "http://www.5963.net/book/19720/" -o ~/my_novels
```

### 设置请求间隔（避免触发反爬）

```bash
# 设置 3 秒间隔
python scripts/novel_crawler.py -u "http://www.5963.net/book/19720/" -d 3
```

### 查看所有选项

```bash
python scripts/novel_crawler.py --help
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-u, --url` | 小说详情页 URL（必需） | 无 |
| `-o, --output` | 输出目录 | `~/downloads` |
| `-d, --delay` | 请求间隔时间（秒） | `2.0` |

## 找到小说 URL 的方法

铅笔小说网没有公开搜索功能，可以通过以下方式找到小说 URL：

1. 访问主页：http://www.5963.net
2. 浏览分类目录（玄幻、武侠、都市等）
3. 点击想要下载的小说
4. 复制浏览器地址栏的 URL

**示例 URL 格式**：
- `http://www.5963.net/book/19720/`

## 注意事项

1. **请求频率**：默认 2 秒间隔，如遇到反爬可适当增加 `-d` 参数
2. **存储位置**：默认保存到 `~/downloads/` 目录
3. **网络稳定**：下载长篇小说可能需要较长时间，请保持网络稳定
4. **版权提醒**：本工具仅用于个人学习和研究，请勿用于商业用途

## 技术实现

- **requests**: HTTP 请求库，支持 Session 保持连接
- **BeautifulSoup**: HTML 解析库，使用 `rel="chapter"` 属性查找章节
- **tldextract**: 域名提取库，确保只访问同域内容
- **反爬处理**: 
  - User-Agent 伪装成 Chrome 浏览器
  - 请求延时（可配置）
  - Session 复用

## 故障排查

### 问题：找不到章节

**原因**：URL 格式不正确

**解决**：确保 URL 格式为 `http://www.5963.net/book/XXXX/`

### 问题：下载失败

**原因**：网络问题或网站反爬

**解决**：增加 `-d` 参数值，如 `-d 5` 表示 5 秒间隔

### 问题：内容乱码

**原因**：编码问题

**解决**：脚本已自动处理编码，如果仍有问题检查网站是否使用特殊编码
