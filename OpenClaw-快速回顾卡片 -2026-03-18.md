## 🎯 一句话总结

**OpenClaw = 个人 AI 助手 + 统一网关 + 多渠道集成**
- 核心口号："EXFOLIATE! EXFOLIATE!" 🦞
- 不是共享 AI，是**个人助手**，单用户单信任边界
- 本地运行，数据本地化

---

## 🏗️ 核心架构（3 层）

```
消息渠道 (20+) 
    ↓
Gateway (ws://127.0.0.1:18789)
    ↓
客户端：CLI/macOS/Web + 节点：iOS/Android
```

### 三大组件
1. **Gateway** - 控制平面（守护进程，WebSocket API）
2. **Clients** - 客户端（macOS/CLI/Web）
3. **Nodes** - 设备节点（iOS/Android/无头）

---

## 🔐 安全三原则

| 原则 | 配置 | 说明 |
|------|------|------|
| 文件权限 | `chmod 700 ~/.openclaw` | 敏感目录保护 |
| 网络绑定 | `bind: "loopback"` | 本地访问，避免暴露 |
| 会话隔离 | `dmScope: "per-channel-peer"` | DM 隔离保护 |

### 安全审计命令
```bash
openclaw security audit
openclaw security audit --deep
openclaw security audit --fix
```

## 📦 安装部署

### 标准安装（推荐）
```bash
curl -fsSL https://openclaw.ai/install.sh | bash
openclaw onboard --install-daemon
```

### 环境要求
- Node.js >= 22（必须）
- pnpm 推荐
- 最低 512MB RAM / 1 核心 / 500MB 磁盘
