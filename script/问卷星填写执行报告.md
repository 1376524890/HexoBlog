## 📞 问题排查

### Q1: 脚本没有自动填写？
**A**: 
1. 检查浏览器控制台是否有错误
2. 确认问卷结构是否匹配
3. 尝试手动刷新页面

### Q2: 提交后没有自动跳转？
**A**: 
1. 可能是最后一份问卷
2. 检查 URL 格式是否正确
3. 可能需要手动复制问卷 ID

### Q3: 问卷星显示"重复提交"？
**A**: 
1. 问卷星限制了同一 IP 的提交次数
2. 使用 Modify Headers 修改请求头
3. 间隔一段时间再提交

### Q4: 如何修改 config 配置？
**A**:
1. 右键 Tampermonkey 图标 → "在编辑器中打开"
2. 找到 `var config = [...]` 部分
3. 修改答案选项
4. 按 Ctrl+S 保存
