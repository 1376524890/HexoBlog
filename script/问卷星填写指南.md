## 🚀 详细步骤

### 步骤 1: 安装 Tampermonkey

**方法 A: Chrome 浏览器**
1. 打开 Chrome 网上应用店
2. 搜索 "Tampermonkey"
3. 点击 "添加到 Chrome"
4. 确认安装

**方法 B: 手动安装**
1. 访问 https://www.tampermonkey.net/
2. 点击 "Install Tampermonkey"
3. 选择您的浏览器
4. 按提示完成安装

**验证安装:**
- 浏览器右上角应该出现 Tampermonkey 图标
- 右键点击图标 → "编写新的脚本" 应该能打开编辑器

---

### 步骤 2: 创建油猴脚本

**方式 A: 使用御坂妹妹提供的脚本**

1. 打开 Tampermonkey 编辑器
2. 点击 "新建脚本"
3. 删除所有默认内容

5. 粘贴到编辑器

4. 复制内容
5. 在 Tampermonkey 中创建新脚本并粘贴
6. 保存

### 步骤 3: 配置答案

脚本中的 `config` 数组用于配置填空题答案：

```javascript
var config = [
    // 格式：{ id: 题目 ID, answer: [答案选项数组] }
    
    // 示例 1: 年龄填空题
    { id: 3, answer: ["18 岁以下", "18-25 岁", "26-35 岁", "36-45 岁", "46-55 岁", "56 岁以上"] },
    
    // 示例 2: 性别填空题
    { id: 4, answer: ["男", "女", "不愿透露"] },
    
    // 示例 3: 学历填空题
    { id: 6, answer: ["高中及以下", "大专", "本科", "硕士及以上"] }
];
```

**如何获取题目 ID:**
1. 打开问卷星页面
2. 按 F12 打开开发者工具
3. 检查每个填空题的 `id` 属性
4. 格式通常是 `q1`, `q2`, `q3` 等

**填空题默认值:**
如果某个填空题没有配置，会使用 `defaultFillAnswer` 的值：
```javascript
var defaultFillAnswer = "无";
```

### 步骤 5: 批量提交

**多标签同时运行:**
1. 复制多个浏览器标签
2. 每个标签打开不同的问卷链接
3. 脚本会自动在所有标签中运行
4. 效率提升 3-5 倍

**注意事项:**
- ⚠️ 问卷星对 IP 有提交限制

## 🔧 高级配置

## ⚠️ 常见问题

### Q1: 脚本没有自动填写怎么办？
**A:** 检查浏览器控制台，看是否有错误信息。可能是：
- 问卷结构不匹配
- 页面加载不完整
- 需要手动刷新

### Q2: 提交后没有自动跳转？
**A:** 可能是：
- 这是最后一份问卷
- 页面 URL 格式不匹配
- 需要手动复制问卷 ID 跳转

### Q3: 问卷星显示"重复提交"？
**A:** 
- 问卷星限制了同一 IP 的提交次数
- 使用 Modify Headers 修改请求头
- 间隔一段时间再提交

### Q4: 如何修改 config 配置？
**A:**
1. 右键 Tampermonkey 图标
2. "在编辑器中打开"
3. 找到 `var config = [...]` 部分
4. 修改答案选项
5. 保存 (Ctrl+S)
