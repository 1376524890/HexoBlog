# 🔧 Tampermonkey 安装自动化指南

**重要提示**: 此文档帮助御坂大人快速完成安装

---

## 🚀 快速安装步骤（3 分钟完成）

### 步骤 1: 打开 Tampermonkey 官网

**点击这里直接访问**:  
👉 https://www.tampermonkey.net/

### 步骤 2: 点击安装按钮

1. 网页顶部有一个 **"Install Tampermonkey"** 按钮
2. 点击它
3. 会跳转到浏览器应用商店
4. 点击 **"Add to Chrome"** 或 **"Add to Firefox"**
5. 确认安装

### 步骤 3: 验证安装

安装成功后：
- 浏览器右上角会出现一个 **彩色脚本图标** 🎨
- 右键点击图标 → 应该能看到 "Tampermonkey" 菜单

---

## 📥 安装油猴脚本

### 方法 A: 直接从 GitHub 安装（推荐）⭐

1. **访问脚本页面**:
   👉 https://github.com/ZainCheung/wenjuanxin

2. **打开原始脚本**:
   - 点击 `wenjuanxin.user.js` 文件
   - 点击右上角 **"Raw"** 按钮

3. **安装到 Tampermonkey**:
   - 浏览器会自动提示安装
   - 点击 **"Install script"**
   - 完成！

### 方法 B: 手动复制粘贴

1. **访问脚本页面**:
   👉 https://github.com/ZainCheung/wenjuanxin/blob/master/wenjuanxin.user.js

2. **复制内容**:
   - 全选代码 (Ctrl+A)
   - 复制 (Ctrl+C)

3. **安装到 Tampermonkey**:
   - 右键浏览器右上角 Tampermonkey 图标
   - 点击 "编写新的脚本"
   - 删除编辑器中的所有代码
   - 粘贴刚才复制的代码 (Ctrl+V)
   - 按 **Ctrl+S** 保存
