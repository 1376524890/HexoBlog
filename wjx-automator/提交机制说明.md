# 问卷星自动填写系统 - 修复版

**问题诊断**：
- 问卷星使用 `show_next_page()` 函数进行分页导航
- 最后一页没有按钮，`show_next_page()` 会自动处理提交
- `submitQuestionnaire()` 函数不存在

---

## 修复内容

### 1. 提交函数 `submit_questionnaire()`

```python
def submit_questionnaire(self) -> bool:
    """
    提交问卷 - 修复版
    
    问卷星提交机制:
    1. 调用 show_next_page() 函数自动提交最后一页
    2. 最后一页没有按钮，show_next_page() 会自动处理
    """
    try:
        self.logger.debug("开始提交问卷")
        
        # 方案 1：直接调用 show_next_page()（问卷星标准提交方式）
        try:
            self.logger.debug("调用 show_next_page()")
            self.driver.execute_script("return show_next_page();")
            time.sleep(3)
            
            if self._check_submission_success():
                self.logger.info("通过 show_next_page() 成功提交")
                return True
        except Exception as e:
            self.logger.warning(f"show_next_page() 调用失败：{e}")
        
        # 方案 2：查找下一页按钮并点击
        try:
            next_btn = self.find_element_safe((By.XPATH, "//a[contains(@onclick, 'show_next_page')]"))
            
            if next_btn:
                self.logger.info("点击下一页按钮")
                self.driver.execute_script("arguments[0].click();", next_btn)
                time.sleep(3)
                
                if self._check_submission_success():
                    self.logger.info("通过点击下一页成功提交")
                    return True
        except Exception as e:
            self.logger.warning(f"点击下一页失败：{e}")
        
        # 方案 3：查找提交按钮
        try:
            submit_btn = self.find_element_safe((By.XPATH, "//div[@id='ctlNext']"))
            
            if submit_btn:
                self.logger.info("点击提交按钮")
                self.driver.execute_script("arguments[0].click();", submit_btn)
                time.sleep(3)
                
                if self._check_submission_success():
                    self.logger.info("通过提交按钮成功提交")
                    return True
        except Exception as e:
            self.logger.warning(f"点击提交按钮失败：{e}")
        
        self.logger.warning("所有提交方式都失败")
        return False
            
    except Exception as e:
        self.logger.error(f"提交失败：{e}")
        return False
```

## 使用说明

1. 打开问卷星页面 https://v.wjx.cn/vm/PhfZxRV.aspx
2. 每页填写后调用 `show_next_page()` 进入下一页
3. 最后一页 `show_next_page()` 会自动处理提交
