# 问卷星自动填写系统 - 修复说明

## 问题诊断

### 原始问题
问卷星页面使用了大量 JavaScript 事件监听，导致：

1. **直接点击 HTML 元素无效** - 使用 Selenium 原生的 `element.click()` 会失败，抛出 `ElementNotInteractableException`
2. **错误的元素选择** - 问卷星的交互元素是 `<a class="jqradio">` 和 `<a class="jqcheckbox">`，而不是 `<input>`
3. **提交按钮触发机制** - 问卷星使用自定义的 JavaScript 函数 `show_next_page()` 和 `submitQuestionnaire()`

### 失败原因
```python
# 错误方式（会失败）
radio = container.find_element(By.XPATH, ".//input[@type='radio']")
radio.click()  # ❌ ElementNotInteractableException
```

## 修复方案

### 核心修复方法

**使用 JavaScript 执行点击**：
```python
# 正确方式（使用 JS）
self.driver.execute_script("arguments[0].click();", element)
```

### 修复的功能模块

#### 1. 单选题填充 (`_fill_single_choice`)

**修复内容**：
- 查找 `div.ui-radio` 中的 `a.jqradio` 元素
- 使用 `execute_script()` 点击
- 支持多种匹配方式（ID、name、value、文本）

```python
def _fill_single_choice(self, container, answer: str) -> Tuple[bool, str]:
    """填充单选题 - 修复版"""
    answer_value = str(answer).strip('"').strip("'")
    
    # 方案 1：通过 input 的 id/name/value 查找对应的 jqradio
    radio_inputs = container.find_elements(By.XPATH, ".//input[@type='radio']")
    
    for radio in radio_inputs:
        # 检查是否匹配
        if matches_condition(radio):
            # 查找同 div 下的 jqradio
            jqradio = radio.find_element(By.XPATH, "..//a[@class='jqradio']")
            # 使用 JavaScript 点击
            self.driver.execute_script("arguments[0].click();", jqradio)
            return True, answer_value
```

#### 2. 多选题填充 (`_fill_multiple_choice`)

**修复内容**：
- 查找 `div.ui-checkbox` 中的 `a.jqcheckbox` 元素
- 遍历所有答案，逐个匹配并点击
- 支持文本匹配和属性匹配

```python
def _fill_multiple_choice(self, container, answers: List[str]) -> Tuple[bool, str]:
    """填充多选题 - 修复版"""
    success_count = 0
    
    # 方案 1：遍历所有 jqcheckbox，根据文本匹配
    jqcheckboxes = container.find_elements(By.XPATH, ".//a[@class='jqcheckbox']")
    
    for jqcheckbox in jqcheckboxes:
        text = jqcheckbox.text.strip()
        
        for answer in answers:
            if answer in text:
                self.driver.execute_script("arguments[0].click();", jqcheckbox)
                success_count += 1
                break
    
    return success_count > 0, f"选择了 {success_count}/{len(answers)} 个选项"
```

#### 3. 量表题填充 (`_fill_scale_question`)

**修复内容**：
- 查找所有 `a.jqradio` 元素
- 根据分数或位置匹配（1-5 分量表）
- 支持文本内容匹配（"非常不同意"、"不同意"等）

```python
def _fill_scale_question(self, container, score: int) -> Tuple[bool, str]:
    """填充量表题（1-5 分）- 修复版"""
    score = int(score)
    
    # 方案 1：通过 jqradio 的文本内容匹配
    jqradios = container.find_elements(By.XPATH, ".//a[@class='jqradio']")
    
    scale_texts = {
        1: ['非常不同意', '非常差', '极差'],
        2: ['不同意', '差', '较少'],
        3: ['一般', '中立', '中等'],
        4: ['同意', '好', '较多'],
        5: ['非常同意', '非常好', '极好']
    }
    
    for jqradio in jqradios:
        text = jqradio.text.strip()
        if any(t in text for t in scale_texts.get(score, [])):
            self.driver.execute_script("arguments[0].click();", jqradio)
            return True, str(score)
```

#### 4. 简答题填充 (`_fill_short_answer`)

**修复内容**：
- 使用 JavaScript 直接设置 `value` 属性
- 触发 `input` 和 `change` 事件（适配 Vue/React 等框架）

```python
def _fill_short_answer(self, container, answer: str) -> Tuple[bool, str]:
    """填充简答题"""
    text_inputs = container.find_elements(By.XPATH, 
        ".//input[@type='text'] | .//textarea")
    
    for text_input in text_inputs:
        # 使用 JavaScript 设置值
        self.driver.execute_script("arguments[0].value = arguments[1];", 
                                  text_input, answer)
        
        # 触发事件
        self.driver.execute_script("""
            arguments[0].dispatchEvent(new Event('input', {bubbles: true}));
            arguments[0].dispatchEvent(new Event('change', {bubbles: true}));
        """, text_input)
        
        return True, "已填写"
```

#### 5. 提交问卷 (`submit_questionnaire`)

**修复内容**：
- 优先调用问卷星的 JavaScript 函数 `show_next_page()` 或 `submitQuestionnaire()`
- 备选方案：查找 `div#ctlNext` 并使用 JS 点击
- 支持分页提交（递归处理下一页）

```python
def submit_questionnaire(self) -> bool:
    """提交问卷 - 修复版"""
    
    # 方案 1：直接调用问卷星的 JavaScript 函数（最可靠）
    try:
        self.driver.execute_script("return show_next_page();")
        time.sleep(2)
        
        if self._check_submission_success():
            return True
    except:
        pass
    
    # 方案 2：调用 submitQuestionnaire()
    try:
        self.driver.execute_script("return submitQuestionnaire();")
        time.sleep(2)
        
        if self._check_submission_success():
            return True
    except:
        pass
    
    # 方案 3：查找提交按钮并使用 JS 点击
    submit_buttons = self.driver.find_elements(By.XPATH,
        "//div[@id='ctlNext'] | //a[contains(@onclick, 'show_next_page')]")
    
    for btn in submit_buttons:
        self.driver.execute_script("arguments[0].click();", btn)
        time.sleep(2)
        
        if self._check_submission_success():
            return True
    
    return False
```

### 新增辅助函数

#### `js_click(element, description)`
使用 JavaScript 点击元素，带详细日志：
```python
def js_click(self, element, description="元素") -> bool:
    """使用 JavaScript 点击元素（关键修复！）"""
    try:
        self.driver.execute_script("arguments[0].click();", element)
        time.sleep(0.3)
        self.logger.debug(f"JavaScript 点击成功：{description}")
        return True
    except Exception as e:
        self.logger.debug(f"JavaScript 点击失败 {description}: {e}")
        return False
```

#### `js_click_by_xpath(xpath, description, timeout)`
通过 XPath 查找并使用 JavaScript 点击：
```python
def js_click_by_xpath(self, xpath: str, description="元素", timeout: int = 10) -> bool:
    """使用 XPath 查找并使用 JavaScript 点击元素（推荐方式）"""
    try:
        element = WebDriverWait(self.driver, timeout).until(
            EC.element_to_be_clickable((By.XPATH, xpath))
        )
        
        self.driver.execute_script("arguments[0].click();", element)
        time.sleep(0.3)
        
        self.logger.debug(f"JavaScript 点击成功 (XPath): {description}")
        return True
    except:
        return False
```

1. **浏览器驱动** - 确保 ChromeDriver 版本与 Chrome 浏览器版本匹配
