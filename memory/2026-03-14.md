## 📚 今日学习重点

### 1. 核心知识回顾

#### OpenClaw 定义
- **一句话**: OpenClaw 是 AI Agent 运行时平台，不是聊天机器人，而是把 AI 连接到真实世界的桥梁
- **四大核心理念**:
  1. Access control before intelligence（访问控制先于智能）⭐⭐⭐⭐⭐
  2. 隐私优先（私有数据保持私有）⭐⭐⭐⭐
  3. 记忆即文件（所有记忆写入 Markdown 文件）⭐⭐⭐⭐⭐
  4. 工具优先（第一类工具而非 skill 包裹）⭐⭐⭐

#### 三层架构
```
Agent Layer (智能层)
     ↓
Gateway Layer (网关层) ← 大脑，不运行 AI 模型，只是调度员
     ↓
Node Layer (节点层) ← 手脚，设备能力
```

#### 四大组件
- **Gateway**: 大脑、路由器
- **Agent**: 执行 AI 任务
- **Session**: 有状态容器
- **Channel**: 协议适配器

#### 御坂网络第一代（多智能体系统）
- 7 个子代理，专业分工
- 权限分级（Level 1-5）
- 本地调度，快速响应
