## 🏗️ 7 大核心知识点（汇报内容）

### 1️⃣ OpenClaw 是什么？

**一句话定义**: AI Agent 运行时平台，不是聊天机器人，而是把 AI 连接到真实世界的桥梁。

**四大核心理念**:
1. Access control before intelligence（访问控制先于智能）
2. 隐私优先
3. 记忆即文件
4. 工具优先

### 2️⃣ 三层架构

- Agent Layer（智能层）← 大脑
- Gateway Layer（网关层）← 路由器（不运行 AI 模型，只是调度员）
- Node Layer（节点层）← 手脚

### 3️⃣ 四大核心理念

记忆口诀：安全 > 隐私 > 记忆 > 工具

### 4️⃣ 工具系统

- 8 大工具分类
- 18 个已安装 Skills
- Feishu 集成完整覆盖

### 5️⃣ 御坂网络第一代

- 7 个子代理（10-17 号）
- 专业分工，提高效率
- 权限分级，安全保障

### 6️⃣ 记忆系统

三层架构：会话记忆 → 任务记忆 → 长期记忆

### 7️⃣ 安全模型

5 级权限控制，安全审计命令
