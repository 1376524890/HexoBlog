### 💡 关键知识点

#### 1. OpenClaw 是什么？
**一句话**：AI Agent 运行时平台，核心是智能网关（Runtime Gateway）
**不是聊天机器人，是把 AI 连接到真实世界的桥梁**

#### 2. 与 ChatGPT 的区别
| 对比项 | ChatGPT | OpenClaw |
|--------|---------|----------|
| 定位 | 聊天机器人 | Agent 运行时平台 |
| 能力 | 生成文本 | 真正执行任务 |
| 记忆 | 会话内临时 | 持久化到磁盘文件 |
| 部署 | 云端 SaaS | 本地部署，数据私有 |

#### 3. 三层架构
```
Agent Layer（智能层）- Main Agent + Subagents
       ↓
Gateway Layer（网关层）- 大脑！路由、策略、调度
       ↓
Node Layer（节点层）- 手脚！摄像头、屏幕、位置
```

#### 4. Agent Loop 核心循环
1. 接收输入 → 2. 构建上下文 → 3. LLM 推理 → 4. 工具执行 → 5. 循环/结束 → 6. 发送响应

#### 5. 子代理系统
- **深度层级**: Depth 0 (Main) → Depth 1 (Sub) → Depth 2 (Leaf)
- **启动方式**: `sessions_spawn()` 或 `/subagents spawn`
- **Announce 机制**: 完成时通知主会话
- **并发控制**: maxConcurrent=8, maxChildrenPerAgent=5

#### 6. 三层记忆架构
- Layer 1: 会话记忆（临时）
- Layer 2: 任务记忆（memory/tasks/）
- Layer 3: 长期记忆（MEMORY.md + 每日日志）
