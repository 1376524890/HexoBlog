## 🛠️ 技术选型清单

### 核心框架
- **Agent 框架**: LangGraph (原生支持循环，状态管理)
- **向量数据库**: ChromaDB (轻量，本地部署)
- **知识图谱**: Neo4j (成熟，支持图查询)
- **嵌入模型**: sentence-transformers (开源，可本地运行)

### 信息搜集
- **RSS 解析**: feedparser
- **爬虫框架**: Scrapy
- **内容清洗**: readability-lxml
- **任务调度**: Celery + Redis

### 监控与评估
- **指标收集**: Prometheus
- **可视化**: Grafana
- **测试框架**: pytest
