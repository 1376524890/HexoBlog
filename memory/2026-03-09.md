## 📋 背景

御坂大人发现 OpenClaw 自身的定时任务系统存在单点故障风险 - 如果 OpenClaw 服务崩溃，所有定时任务都会失效！

**解决方案**：将关键任务迁移到系统级 cron，即使 OpenClaw 崩溃也能继续运行。

---

## 🔄 迁移的任务清单

| 任务 | OpenClaw Cron | 系统 Cron | 状态 |
|------|--------------|----------|------|
| 自动备份 (每 6 小时) | ✅ | ✅ 新安装 | 双保险 |
| 清理旧备份 (每天 12:30) | ✅ | ✅ 新安装 | 双保险 |
| 服务恢复检查 (每 30 分钟) | ❌ | ✅ 新任务 | 新增 |

## 📁 创建的脚本

### 1. `/home/claw/.openclaw/bin/autobackup.sh`
每 6 小时备份工作区，保留最近 7 天的备份。

**功能**：
- 压缩备份 `~/.openclaw/workspace`
- 记录到 `autobackup.log`
- 自动清理 7 天前的备份

### 2. `/home/claw/.openclaw/bin/backup-cleanup.sh`
每天 12:30 清理旧备份和日志文件。

**功能**：
- 删除 7 天前的 `.tar.gz` 文件
- 删除 30 天前的 `.log` 文件
- 执行 `git gc --prune=now` 清理 Git 垃圾

### 3. `/home/claw/.openclaw/bin/openclaw-recover.sh`
每 30 分钟检查 OpenClaw 服务状态，异常时自动重启。

**功能**：
- 检查 OpenClaw 进程是否存在
- 如果未运行，自动重启 gateway
- 记录系统资源状态（磁盘、内存）
- 记录到 `openclaw-recover.log`

## ⏰ 系统 Cron 作业配置

御坂大人，现在系统级 cron 已安装这些任务：

```bash
# 每 6 小时备份一次 (0,6,12,18 点)
0 */6 * * * /home/claw/.openclaw/bin/autobackup.sh >> /home/claw/.openclaw/backup/cron-autobackup.log 2>&1

# 每天 12:30 清理旧备份
30 12 * * * /home/claw/.openclaw/bin/backup-cleanup.sh >> /home/claw/.openclaw/backup/cron-cleanup.log 2>&1

# 每 30 分钟检查 OpenClaw 服务状态并恢复
*/30 * * * * /home/claw/.openclaw/bin/openclaw-recover.sh >> /home/claw/.openclaw/backup/cron-recover.log 2>&1
```

**查看当前 cron 任务**：
```bash
crontab -l
```

## 🛡️ 安全冗余设计

### 三级备份机制
1. **OpenClaw 自身定时任务** - 主用，功能丰富
2. **系统级 cron 备份** - 备用，独立运行
3. **手动备份** - 御坂大人随时触发

### 服务恢复机制
- OpenClaw 崩溃 → 系统 cron 每 30 分钟检测到
- `openclaw-recover.sh` 自动重启 gateway
- 无需人工干预

### 日志隔离
- `autobackup.log` - 备份日志
- `cron-autobackup.log` - 系统级备份日志
- `cron-cleanup.log` - 系统级清理日志
- `cron-recover.log` - 服务恢复日志

## 🔧 使用说明

### 检查服务状态
```bash
systemctl status cron
```

### 查看 cron 任务
```bash
crontab -l
```

### 手动执行备份
```bash
/home/claw/.openclaw/bin/autobackup.sh
```

### 查看最近日志
```bash
tail -20 ~/.openclaw/backup/cron-*.log
```

### 临时禁用任务
```bash
# 编辑 cron
crontab -e

# 注释掉不需要的行
# 0 */6 * * * /home/claw/.openclaw/bin/autobackup.sh
```

### 重新启用任务
```bash
crontab -e
# 取消注释
```
