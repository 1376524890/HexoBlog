### 🔍 Git 操作细节

**遇到的问题**：
1. 远程仓库有更新，需要先 pull
2. 本地有未提交文件，需要 stash
3. 使用 `--rebase` 方式合并

**解决步骤**：
```bash
# 暂存未提交文件
git stash

# 拉取远程更新
git pull origin master --rebase

# 恢复暂存文件
git stash pop

# 推送发布
git push origin master
```

### 📊 技术要点记录

#### 系统提示词五层结构
```
Layer 1: 基础身份 (名称/描述/Emoji/身份声明)
Layer 2: 工具描述 (JSON Schema 参数说明)
Layer 3: 运行时信息 (时间/工作目录/环境变量)
Layer 4: 安全提示 (沙箱边界/权限限制/禁止行为)
Layer 5: 格式说明 (工具调用格式/回复规则)
```

#### Agent Loop 8 步流程
```
1. Gateway 接收 Channel 消息
2. 构建上下文 (System Prompt + 历史 + 工具)
3. LLM 推理 (模型决定回复或工具调用)
4. 工具执行 (权限检查→调用→返回结果)
5. 追加结果到历史
6. 判断是否继续 (循环或完成)
7. 发送响应
8. 更新 Session
```

#### Session 生命周期
```
创建 → 活跃 → 压缩 → 归档/删除
```

**压缩机制**：
- 触发条件：token 使用量 > 60%
- 保留最近 5 条原始消息
- LLM 总结历史摘要
