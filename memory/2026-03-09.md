### 功能特点

1. **多服务支持**：
   - r.jina.ai（主要）⭐
   - markdown.new（回退）
   - defuddle.md（备选）

2. **智能总结**：自动提取标题和摘要

3. **统计信息**：显示行数和单词数

4. **优雅提示**：御坂风格的错误提示

### 使用示例

```python
# 方式 1: 使用 web_fetch 工具（推荐）

  "url": "https://r.jina.ai/https://github.com/openclaw/openclaw",
  "extractMode": "markdown"
})

# 方式 2: 使用 Python 脚本
python3 ~/.openclaw/skills/web-markdown-search/web_markdown_search.py https://example.com
```
