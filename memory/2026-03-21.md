## 🎯 核心知识点总结

### 1. OpenClaw 是什么？

**一句话**: OpenClaw 是 AI Agent 运行时平台，不是聊天机器人，而是把 AI 连接到真实世界的桥梁。

**官方定义**: 自托管网关（self-hosted gateway），将您最喜欢的聊天应用（WhatsApp、Telegram、Discord、iMessage 等）与 AI 编码智能体连接起来。

### 2. 四大核心理念（⭐⭐⭐⭐⭐）

1. **Access control before intelligence**（访问控制先于智能）
2. **隐私优先**：私有数据保持私有
3. **记忆即文件**：所有记忆写入磁盘 Markdown 文件
4. **工具优先**：第一类工具而非 skill 包裹

**记忆口诀**: 安全 > 隐私 > 记忆 > 工具

### 3. 三层架构

```
Agent Layer（智能层）
  ↓
Gateway Layer（网关层）← 大脑！不运行 AI，只是调度员
  ↓
Node Layer（节点层）← 手脚
```

**四大核心组件**:
- **Gateway** - 大脑、路由器，不运行 AI 模型
- **Agent** - 执行 AI 任务，身份 + 配置 + 状态 + 运行时
- **Session** - 有状态容器，消息历史、上下文、工具状态
- **Channel** - 协议适配器（Telegram、Discord、飞书等）

```
Layer 1: 会话记忆（Session Memory）
   ↓

   ↓
Layer 3: 长期记忆（Long-term Memory）
```

**记忆工具**: `memory_search`, `memory_get`
