## 🎯 集成到 OpenClaw

### 1️⃣ 使用场景

当御坂美琴一号被问及问题时：

1. **先搜索工作空间** - 查找相关文档
2. **提取关键信息** - 从搜索结果中获取上下文
3. **基于证据回答** - 用文档内容支持回答

### 2️⃣ 示例代码

```python
from skills.local-knowledge-base.src.searcher import KnowledgeSearcher

def answer_with_context(question: str) -> str:
    # 搜索相关问题
    searcher = KnowledgeSearcher()
    results = searcher.search(question, top_k=3)
    
    # 构建上下文
    context = []
    for result in results:
        doc = result['document']
        context.append(f"文档：{doc['path']}")
        context.append(f"内容：{doc['content'][:200]}")
        context.append("-" * 40)
    
    # 使用上下文回答问题
    full_prompt = f"""
问题：{question}

相关文档：
{''.join(context)}

请根据以上文档内容回答问题：
"""
    
    return full_prompt
```
