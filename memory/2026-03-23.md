## 🎯 核心知识点（30-40 分钟汇报大纲）

### 1️⃣ OpenClaw 是什么？（5 分钟）

**一句话定义**: OpenClaw 是 **AI Agent 运行时平台**，核心是**智能网关（Runtime Gateway）**。  
**核心理念**:
- Access control before intelligence（访问控制先于智能）
- 隐私优先：私有数据保持私有
- 记忆即文件：所有记忆写入 Markdown 文件
- 工具优先：第一类工具而非 skill 包裹

### 2️⃣ 核心架构（10 分钟）

**三层架构**（脑→路由→手）:
- **Agent Layer**: 智能层（脑）- 主 Agent、子代理、ACP 编码代理
- **Gateway Layer**: 网关层（路由）- 控制平面、策略层、路由、身份认证
- **Node Layer**: 节点层（手）- 远程执行、设备能力

**四大组件**:
- **Gateway**: 大脑、路由器（不运行 AI 模型）
- **Agent**: 执行 AI 任务（主动决策）
- **Session**: 有状态容器（维护会话）
- **Channel**: 协议适配器（支持 15+ 平台）

**Agent Loop**: 接收输入 → 构建上下文 → LLM 推理 → 工具执行 → 循环/结束 → 发送响应

### 3️⃣ 工具与技能系统（8 分钟）

**8 大工具分类**: Runtime、Filesystem、Session、Memory、Web、UI、Node、Messaging
