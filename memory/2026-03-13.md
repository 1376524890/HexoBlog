## 🎯 核心知识点掌握

### 一、OpenClaw 定义（必背）

**一句话**: OpenClaw 是 **AI Agent 运行时平台**，不是聊天机器人，而是把 AI 连接到真实世界的桥梁。

**四大核心理念**（⭐⭐⭐⭐⭐）:
1. ✅ **Access control before intelligence**（访问控制先于智能）
2. ✅ **隐私优先**：私有数据保持私有
3. ✅ **记忆即文件**：所有记忆写入 Markdown 文件
4. ✅ **工具优先**：第一类工具而非 skill 包裹

**对比 ChatGPT**:
- ChatGPT: 聊天机器人，生成文本，云端 SaaS
- OpenClaw: Agent 运行时，执行任务，本地部署

---

### 二、三层架构（必背）

```
Agent Layer（智能层）
  ↓
Gateway Layer（网关层）← 大脑，不运行 AI 模型
  ↓
Node Layer（节点层）← 手脚
```

**四大组件**:
- Gateway: 大脑、路由器，不运行 AI 模型，只是调度员
- Agent: 执行 AI 任务
- Session: 有状态容器
- Channel: 协议适配器

**Agent Loop**:
1. 接收输入
2. 构建上下文
3. LLM 推理
4. 工具执行
5. 循环或结束
6. 发送响应
