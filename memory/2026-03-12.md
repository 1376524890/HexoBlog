#### 1. 核心架构学习

#### 2. 四大核心组件
- **Gateway**: 控制平面、策略层、路由，不运行 AI 模型，只是调度员
- **Agent**: 实际执行 AI 任务的实例
- **Session**: 有状态的会话容器
- **Channel**: 协议适配器

#### 3. Agent Loop 核心循环
6 个步骤：接收输入 → 构建上下文 → LLM 推理 → 工具执行 → 循环或结束 → 发送响应

#### 4. 记忆系统
- **双重架构**: 短期记忆 (memory/YYYY-MM-DD.md) + 长期记忆 (MEMORY.md)
- **记忆工具**: memory_search, memory_get
- **特性**: 纯 Markdown、向量检索、混合搜索、自动压缩、时间衰减
