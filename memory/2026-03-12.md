### 2. 掌握的核心知识点

#### 核心定义
- OpenClaw 是 AI Agent 运行时平台，不是聊天机器人
- 核心：智能网关（Runtime Gateway）
- 四大核心理念：
  1. Access control before intelligence（访问控制先于智能）⭐⭐⭐⭐⭐
  2. 隐私优先
  3. 记忆即文件
  4. 工具优先

#### 三层架构
- Agent Layer（智能层）：Main Agent、Subagents、ACP Agents
- Gateway Layer（网关层）：大脑，不运行 AI，只是调度员
- Node Layer（节点层）：手脚，设备能力

#### 四大组件
- Gateway：大脑、路由器
- Agent：执行 AI 任务
- Session：有状态容器
- Channel：协议适配器

#### 御坂网络第一代（7 个子代理）
- 10 号：通用代理（general-agent）
- 11 号：代码执行者（code-executor）

- 14 号：文件管理器（file-manager）
- 15 号：系统管理员（system-admin）

#### 记忆系统
- 三层架构：会话记忆→任务记忆→长期记忆
- 文件位置：MEMORY.md、memory/YYYY-MM-DD.md
