## 📚 核心知识体系

### 1️⃣ OpenClaw 是什么？

**一句话定义**: OpenClaw 是 **AI Agent 运行时平台**，核心是**智能网关（Runtime Gateway）**。  
**核心理念**:
- Access control before intelligence（访问控制先于智能）⭐⭐⭐⭐⭐
- 隐私优先：私有数据保持私有
- 记忆即文件：所有记忆写入 Markdown 文件
- 工具优先：第一类工具而非 skill 包裹

### 2️⃣ 三层架构（必讲）

```
Agent Layer（智能层）← 脑
     ↓
Gateway Layer（网关层）← 路由（核心！）
     ↓
Node Layer（节点层）← 手
```

**四大组件**:
- **Gateway**: 大脑、路由器（不运行 AI 模型）
- **Agent**: 执行 AI 任务（主动决策）
- **Session**: 有状态容器
- **Channel**: 协议适配器（15+ 平台）

**Agent Loop**: 接收输入 → 构建上下文 → LLM 推理 → 工具执行 → 循环/结束 → 发送响应

### 3️⃣ 工具系统（8 大分类）

- Runtime: `exec`, `process`, `gateway`
- Filesystem: `read`, `write`, `edit`
- Session: `sessions_list`, `sessions_spawn`
- Memory: `memory_search`, `memory_get`
- Web: `web_search`, `web_fetch`
- UI: `browser`, `canvas`
- Node: `nodes`
- Messaging: `message`
