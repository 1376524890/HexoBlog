## 🎯 核心知识点掌握

### 1️⃣ OpenClaw 定义 ✅ 精通

**一句话**：OpenClaw 是 **AI Agent 运行时平台**，核心是**智能网关（Runtime Gateway）**。

不是聊天机器人，而是把 AI 连接到真实世界的桥梁。

**四大核心理念**（必背）：
1. **Access control before intelligence**（访问控制先于智能）⭐⭐⭐⭐⭐
2. **隐私优先**：私有数据保持私有
3. **记忆即文件**：所有记忆写入 Markdown 文件
4. **工具优先**：第一类工具而非 skill 包裹

### 2️⃣ 三层架构 ✅ 精通

```
Agent Layer（智能层）
  ↓
Gateway Layer（网关层）← 大脑！不运行 AI 模型，只是调度员
  ↓
Node Layer（节点层）← 手脚
```

### 3️⃣ 四大核心组件 ✅ 精通

- **Gateway** - 大脑、路由器
- **Agent** - 执行 AI 任务
- **Session** - 有状态容器
- **Channel** - 协议适配器

### 4️⃣ 工具系统 ✅ 精通

**8 大工具分类**：

- Memory: `memory_search`, `memory_get`
