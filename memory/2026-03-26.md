## 📚 核心知识点

### 1. 定义（精通）⭐⭐⭐⭐⭐
> OpenClaw 是 AI Agent 运行时平台，核心是智能网关（Runtime Gateway）。  
> 它不是聊天机器人，而是把 AI 连接到真实世界的桥梁。

### 2. 四层理念（精通）⭐⭐⭐⭐⭐
1. **Access control before intelligence**（访问控制先于智能）
2. **隐私优先**：私有数据保持私有
3. **记忆即文件**：所有记忆写入 Markdown 文件
4. **工具优先**：第一类工具而非 skill 包裹

### 3. 三层架构（精通）⭐⭐⭐⭐⭐
```
Agent Layer（智能层）← 脑
    ↓
Gateway Layer（网关层）← 路由
    ↓
Node Layer（节点层）← 手
```

### 4. Gateway 架构（精通）⭐⭐⭐⭐⭐
- 控制平面：WebSocket，JSON Schema 验证
- 事件类型：agent、chat、presence、health、heartbeat、cron
- 安全机制：设备身份认证、配对流程、签名验证
- 会话管理：dmScope 配置、session 维护

### 5. Session 管理（精通）⭐⭐⭐⭐⭐
- Key 格式：agent:agentId:main / direct / group / channel / cron / hook / node
- dmScope: main / per-peer / per-channel-peer / per-account-channel-peer
- 维护：pruneAfter、maxEntries、rotateBytes、highWaterBytes

### 6. 工具系统（精通）⭐⭐⭐⭐⭐
- 8 大分类：Runtime、Filesystem、Session、Memory、Web、UI、Node、Messaging
- Feishu 集成：doc、chat、drive、wiki、bitable

### 7. 记忆系统（精通）⭐⭐⭐⭐⭐
- 三层架构：会话记忆 → 任务记忆 → 长期记忆
- WAL Protocol：STOP → WRITE → THEN
- 触发条件：修正、专有名词、偏好、决策、编辑、数值

### 8. 多智能体系统（精通）⭐⭐⭐⭐⭐
- 本尊 + 一号 + 7 个子代理（10-17 号）
- 四角色闭环：Planner → Executor → Reviewer → Patrol

### 9. 安全模型（精通）⭐⭐⭐⭐⭐
- 5 级权限：Level 5 (主 Agent) → Level 1 (只读)
- 安全机制：设备配对、沙箱隔离、工具控制、审计日志
- 安全命令：`openclaw security audit` / `--deep` / `--fix`

### 10. Cron vs Heartbeat（熟练）⭐⭐⭐⭐
- Heartbeat：批量处理、需要上下文、时间可漂移
- Cron：精确时间、历史隔离、不同模型、一次性提醒

## 🎯 核心洞见

1. ✅ **不是聊天机器人**，而是能真正执行任务的 Agent 平台
2. ✅ **记忆即文件**，所有记忆持久化到磁盘，不丢失
3. ✅ **安全第一**，多层权限控制和审计日志
4. ✅ **模块化设计**，Skills 和 Channels 独立可替换
5. ✅ **多智能体协作**，专业分工，效率更高
6. ✅ **自托管部署**，数据完全掌控在用户手中
7. ✅ **跨平台支持**，一个 Gateway 服务多个聊天应用
8. ✅ **路由灵活**，支持单多 Agent、单多账户、多角色路由
9. ✅ **模型中立**，支持本地模型（vllm）和远程 API
10. ✅ **开源许可**，MIT 许可，社区驱动

**新增洞见（2026-03-23 深化）**:
1. ✅ **Gateway 不是 AI 模型**，只是调度员和控制平面
2. ✅ **Session 是关键状态**，所有会话状态存储在 sessions.json
3. ✅ **安全 DM 模式必要**：多用户场景必须启用 `dmScope: per-channel-peer`
4. ✅ **Session 维护重要**：定期清理防止磁盘膨胀
5. ✅ **工具优先设计**：工具是第一类能力，不是 skill 包裹
6. ✅ **Cron 与 Heartbeat 互补**: Cron 精确定时，Heartbeat 批量处理
7. ✅ **WAL Protocol 确保持久化**: 先写后读，避免记忆丢失
8. ✅ **四角色闭环体系**: Planner-Executor-Reviewer-Patrol 完整闭环
