## 🏗️ 系统架构 (1 分钟)

```
聊天应用 → Gateway → AI Agent
              ↓
         CLI / Web UI / 节点
```

**Gateway 是核心** - 唯一事实来源:
- ✅ 维护通道连接
- ✅ 通过 WebSocket 暴露 API
- ✅ 处理认证、路由、会话管理
- ✅ 内置 cron 调度器

**控制面板**: `http://127.0.0.1:18789`

## 🛠️ 工具系统 (1 分钟)

**20+ 核心工具**:

| 工具组 | 代表工具 |
|--------|----------|
| group:fs | read, write, edit |
| group:sessions | sessions_list, sessions_spawn |
| group:web | web_search, web_fetch |
| group:memory | memory_search, memory_get |
| group:nodes | nodes (控制节点) |
| group:automation | cron (定时任务) |

**工具策略**:
- **profile**: minimal/coding/messaging/full
- **精细控制**: allow/deny
- **技能系统**: Skills 教代理如何使用工具

## 🔐 安全要点 (1 分钟)

### ⚠️ 关键安全机制

1. **DM 安全模式**
   - `dmScope: "per-channel-peer"` (推荐)
   - **防止隐私泄露**！

2. **认证令牌**
   - 所有连接必须签名 nonce

3. **沙箱隔离**
   - Docker 容器隔离
   - `off` | `non-main` | `all`

4. **工具权限**
   - 精细的 allow/deny 控制

### 🚨 安全警告

> 如果代理接收多人的 DM，**必须**启用安全 DM 模式！

## 📋 快速命令 (30 秒)

```bash
# 状态检查
openclaw status

# 会话管理
openclaw sessions list

# Cron 任务
openclaw cron list
openclaw cron add --name "任务" --cron "0 7 * * *"

# 配置管理
openclaw config get
openclaw configure

# 启动网关
openclaw gateway --port 18789
```
