## 🎯 核心知识点回顾

### 1. OpenClaw 是什么？

**一句话定义**:
> **OpenClaw 是 AI Agent 运行时平台**，不是聊天机器人，而是把 AI 连接到真实世界的桥梁。通过单一 Gateway 控制平面，可以统一管理 20+ 消息渠道，实现多智能体协作。

### 2. 四大核心理念（必背）⭐⭐⭐⭐⭐

1. **Access control before intelligence**（访问控制先于智能）
2. **隐私优先**：私有数据保持私有
3. **记忆即文件**：所有记忆写入 Markdown 文件
4. **工具优先**：第一类工具而非 skill 包裹

### 3. 三层架构

```
Agent Layer（智能层）
  ↓
Gateway Layer（网关层）← 大脑！不运行 AI，只是调度员
  ↓
Node Layer（节点层）← 手脚
```

### 4. 御坂网络第一代

1 个主 Agent + 7 个子代理（10-17 号），专业分工协作

### 5. 三层记忆架构

会话记忆 → 任务记忆 → 长期记忆（MEMORY.md）

### 6. 安全模型

5 级权限控制（Level 1-5），安全审计命令
