```
┌─────────────────┐
```

## 🧠 记忆系统

**三层架构**:
1. 会话记忆（Session Memory）
2. 任务记忆（Task Memory）
3. 长期记忆（MEMORY.md + memory/YYYY-MM-DD.md）

**安全规则**:
- ✅ 用 `trash` 而不是 `rm`
- ✅ 操作前备份
- ✅ 立即 `git add` + `git commit`

## 🛡️ 安全模型

**5 级权限**:
- Level 5: 主 Agent（完全权限）
- Level 4: 可信子 Agent（需批准）
- Level 3: 标准子 Agent
- Level 2: 受限子 Agent
- Level 1: 只读子 Agent

**审计命令**:
```bash
openclaw security audit --deep
```
