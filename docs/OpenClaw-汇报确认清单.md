## ⚠️ 注意事项

### 技术细节确认

1. **Gateway 默认端口**：18789
2. **WebSocket 协议**：typed API
3. **工具分组**：`group:*` 快捷方式
4. **会话密钥格式**：`agent:<agentId>:...`
5. **技能加载优先级**：workspace > local > bundled

### 安全注意事项

1. **安全 DM 模式**：推荐 `per-channel-peer`
2. **工具权限控制**：使用 `tools.profile`
3. **设备配对**：需要明确批准
4. **沙箱机制**：不可信工具需沙箱运行
5. **技能审核**：第三方技能需 vetting
