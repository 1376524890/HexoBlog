## 🎯 7 大核心知识点（汇报必讲）

### 1️⃣ OpenClaw 是什么？⭐⭐⭐⭐⭐

**一句话定义**:
> **OpenClaw 是 AI Agent 运行时平台**，不是聊天机器人，而是把 AI 连接到真实世界的桥梁。

**核心区别**:
- ChatGPT = 聊天机器人（生成文本）
- OpenClaw = Agent 运行时平台（真正执行任务）

**核心特点**:
- ✅ 自托管：运行在自己的机器上，数据私有
- ✅ 多通道：一个 Gateway 服务多个聊天应用
- ✅ Agent 原生：内置工具使用、记忆、多 Agent 路由
- ✅ 开源：MIT 许可，社区驱动

**掌握状态**: ✅ 精通
