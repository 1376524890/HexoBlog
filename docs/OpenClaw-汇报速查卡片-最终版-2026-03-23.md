## 🏗️ 三层架构（必讲！）

```
Agent Layer（智能层）← 脑
     ↓
Gateway Layer（网关层）← 路由（核心！）
     ↓
Node Layer（节点层）← 手
```

**记忆口诀**: 脑→路由→手

### 四大核心组件

| 组件 | 作用 | 关键点 |
|------|------|--------|
| Gateway | 大脑、路由器 | 不运行 AI 模型，只是调度员 |
| Agent | 执行 AI 任务 | 模型拥有决策权 |
| Session | 有状态容器 | 消息历史、上下文、工具状态 |
| Channel | 协议适配器 | 支持 15+ 个聊天平台 |

### Agent Loop（6 步）

1. 接收输入
2. 构建上下文
3. LLM 推理
4. 工具执行
5. 循环或结束
6. 发送响应
