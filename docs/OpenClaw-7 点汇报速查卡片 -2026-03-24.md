## 🏗️ 三层架构（必讲！）

### 架构总览

```
┌─────────────────────────────────────────────────────────┐
│              Agent Layer（智能层）                        │
│  - Main Agent、Subagents、ACP Agents                    │
│  - 执行 AI 任务，拥有决策权                             │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│           Gateway Layer（网关层）← 大脑！                  │
│  - 控制平面、策略层、路由                               │
│  - 身份认证、工具策略、会话管理                         │
│  - 频道适配器（15+ 个聊天平台）                          │
│  ⚠️ Gateway 本身不运行 AI 模型，只是调度员                  │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│              Node Layer（节点层）← 手脚                   │
│  - 远程执行表面                                         │
│  - 设备能力（摄像头、屏幕、通知、位置）                 │
│  - macOS companion app, iOS/Android nodes               │
└─────────────────────────────────────────────────────────┘
```

**记忆口诀**: 智能层（脑）→ 网关层（路由）→ 节点层（手）

### 核心组件

#### Gateway 层
- **控制平面**: WebSocket 连接管理，JSON Schema 验证
- **事件类型**: `agent`、`chat`、`presence`、`health`、`heartbeat`、`cron`
- **安全机制**: 设备身份、配对流程、签名验证

#### Session 管理
- **Key 格式**:
  - 直接聊天：`agent:<agentId>:main` / `agent:<agentId>:direct:<peerId>`
  - 群组聊天：`agent:<agentId>:<channel>:group:<id>`
  - 频道聊天：`agent:<agentId>:<channel>:channel:<id>`
  - Cron 任务：`cron:<jobId>`
- **dmScope 配置**:
  - `per-channel-peer`: 多用户推荐（安全 DM 模式）
  - `per-account-channel-peer`: 多账户推荐
- **维护策略**: 清理 45 天前、800 个会话上限、20MB 轮换

#### Channel 支持
- **15+ 平台**: Telegram（最快）、Discord、Slack、WhatsApp、飞书、iMessage 等

## 🔧 四大核心组件

### 1. Gateway（网关）⭐⭐⭐⭐⭐
- **作用**: 大脑、路由器
- **关键点**: 不运行 AI 模型，只是调度员
- **默认端口**: 18789

### 2. Agent（智能体）
- **作用**: 执行 AI 任务
- **Agent Loop**: 接收输入 → 构建上下文 → LLM 推理 → 工具执行 → 循环/结束 → 发送响应

### 3. Session（会话）
- **作用**: 有状态容器
- **内容**: 消息历史、上下文信息、工具状态、临时决策

### 4. Channel（频道）
- **作用**: 协议适配器
- **支持平台**: 15+ 个（Telegram、Discord、飞书、WhatsApp 等）
