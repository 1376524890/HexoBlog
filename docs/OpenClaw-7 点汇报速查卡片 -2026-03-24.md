## 6️⃣ 三层记忆架构（必讲）⭐⭐⭐⭐⭐

```
会话记忆 → 任务记忆 → 长期记忆
```

**WAL Protocol（写后读协议）**：
- STOP — 不要立即回复
- WRITE — 更新记忆文件
- THEN — 回复用户

**触发条件**：修正、专有名词、偏好、决策、编辑、数值
