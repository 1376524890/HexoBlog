## 🏗️ 三层架构（必画）

```
Agent Layer（智能层）← 大脑的执行者
  ↓
Gateway Layer（网关层）← 中枢！不运行 AI，只是调度员
  ↓
Node Layer（节点层）← 手脚
```

**四大组件**:
- **Gateway**: 控制平面、路由、策略
- **Agent**: 执行 AI 任务
- **Session**: 有状态容器
- **Channel**: 协议适配器

## ❓ 常见问题（Q&A）

**Q1: OpenClaw 和 ChatGPT 的区别？**  
A1: ChatGPT 是聊天机器人，OpenClaw 是 Agent 运行时，能真正执行任务

**Q2: 数据安全性如何保障？**  
A2: 自托管、三层权限模型、审计日志、沙箱隔离

A5: 支持
