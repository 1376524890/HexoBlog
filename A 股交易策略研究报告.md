## 二、交易策略详解

### 2.1 均线交叉策略 (MA Crossover)

#### 原理
- **金叉买入**: 短期均线上穿长期均线
- **死叉卖出**: 短期均线下穿长期均线

#### 常用参数
```python
# 常用均线组合
MA_SHORT = 5      # 5 日均线
MA_MEDIUM = 10    # 10 日均线  
MA_LONG = 20      # 20 日均线
```

#### 策略信号
| 信号 | 条件 | 操作 |
|------|------|------|
| 金叉 | MA5 > MA10 且 MA5 从前一日 < MA10 | 买入 |
| 死叉 | MA5 < MA10 且 MA5 从前一日 > MA10 | 卖出 |
| 多头 | MA5 > MA10 > MA20 | 持有 |
| 空头 | MA5 < MA10 < MA20 | 观望 |

#### 优点
- 简单易懂，趋势明显
- 适合趋势行情
- 延迟较低

#### 缺点
- 震荡行情容易来回打脸
- 滞后性

---

### 2.2 MACD 指标策略

#### 原理
MACD (Moving Average Convergence Divergence) 是趋势跟踪动量指标

#### 参数设置
```python
# MACD 标准参数
FAST = 12       # 快线周期
SLOW = 26       # 慢线周期
SIGNAL = 9      # 信号线周期
```

#### 计算公式
```
EMA12 = 12 日指数移动平均线
EMA26 = 26 日指数移动平均线
DIF = EMA12 - EMA26   # 快线
DEA = DIF 的 9 日 EMA  # 慢线
MACD = (DIF - DEA) × 2  # 柱状图
```

#### 交易信号
| 信号 | 条件 | 操作 |
|------|------|------|
| 金叉 | DIF 上穿 DEA | 买入 |
| 死叉 | DIF 下穿 DEA | 卖出 |
| 顶背离 | 价格创新高，MACD 未创新高 | 卖出 |
| 底背离 | 价格创新低，MACD 未创新低 | 买入 |
| 零轴上方金叉 | 强势金叉 | 强烈买入 |
| 零轴下方金叉 | 弱势金叉 | 谨慎买入 |

#### 优点
- 趋势判断准确
- 可识别背离
- 适合中线交易

#### 缺点
- 滞后性较强
- 震荡市效果差

---

### 2.3 RSI 指标策略

#### 原理
RSI (Relative Strength Index) 相对强弱指标，衡量买卖力量对比

#### 参数设置
```python
RSI_PERIOD = 14  # 常用 14 日
```

#### 计算公式
```
RS = 平均涨幅 / 平均跌幅
RSI = 100 - (100 / (1 + RS))
```

#### 交易信号
| 区域 | RSI 值 | 含义 | 操作 |
|------|-------|------|------|
| 超买区 | > 70 | 买入过度，可能回调 | 卖出 |
| 强势区 | 50-70 | 多头占优 | 持有/买入 |
| 弱势区 | 30-50 | 空头占优 | 观望 |
| 超卖区 | < 30 | 卖出过度，可能反弹 | 买入 |

#### 进阶用法
- **RSI 背离**: RSI 与价格走势背离时反转信号强
- **RSI 形态**: RSI 形成头肩顶、双底等形态

#### 优点
- 快速识别超买超卖
- 背离信号可靠
- 适合短线交易

#### 缺点
- 单边行情容易钝化
- 需结合其他指标

### 2.4 Bollinger Bands 布林带策略

#### 原理
利用标准差计算上下轨，判断价格波动范围

#### 参数设置
```python
BB_PERIOD = 20     # 移动平均周期
BB_STD = 2.0       # 标准差倍数
```

#### 计算公式
```
Middle = N 日简单移动平均 (SMA)
Upper = Middle + N 日标准差 × 2
Lower = Middle - N 日标准差 × 2
Bandwidth = (Upper - Lower) / Middle
```

#### 交易信号
| 信号 | 条件 | 操作 |
|------|------|------|
| 触上轨 | 价格触及或突破上轨 | 卖出 |
| 触下轨 | 价格触及或跌破下轨 | 买入 |
| 缩口 | Bandwidth 持续缩小 | 等待突破 |
| 张口 | Bandwidth 急剧扩大 | 趋势启动 |
| 中轨支撑 | 价格回踩中轨未破 | 买入 |
| 中轨压力 | 价格反弹中轨受阻 | 卖出 |

#### 布林带形态
- **喇叭口**: 上下轨扩张，趋势强烈
- **收口**: 上下轨收缩，即将变盘
- **行走**: 价格沿上轨/下轨运行，趋势明显

#### 优点
- 直观显示波动率
- 可识别突破信号
- 适合震荡 + 趋势行情

#### 缺点
- 需要配合其他指标
- 单边行情信号少

## 三、综合策略建议

### 3.1 策略组合

对于 **10,000 元小资金**，建议采用**多指标共振**策略：

#### 核心策略：MA + MACD + RSI
```
买入条件 (需同时满足):
1. MA5 上穿 MA10 (金叉)
2. MACD DIF 上穿 DEA (金叉)
3. RSI 从超卖区 (30 以下) 回到 50 以上
4. 股价在布林带下轨附近或中轨上方

卖出条件 (满足任一):
1. MA5 下穿 MA10 (死叉)
2. MACD DIF 下穿 DEA (死叉)
3. RSI 进入超买区 (>70) 并拐头向下
4. 达到止盈 (+15%) 或止损 (-5%)
```

### 3.2 资金管理方案

#### 资金分配 (10,000 元)
```
总资金：10,000 元

方案 A: 分散投资
- 股票 1: 5,000 元 (50%)
- 股票 2: 5,000 元 (50%)
- 持仓上限：2 只

方案 B: 集中投资
- 股票 1: 3,000 元 (30%)
- 股票 2: 2,000 元 (20%)
- 现金储备：5,000 元 (50%)
- 持仓上限：2 只
```

#### 单笔交易规则
- **单笔金额**: 3,000-5,000 元 (30%-50%)
- **止损**: -5% (严格执行)
- **止盈**: +15% (达到目标分批止盈)
- **最大亏损**: 单笔最大亏损 250 元 (5,000×5%)
- **最大盈利**: 单笔最大盈利 750 元 (5,000×15%)

### 3.3 选股标准

#### 基本面筛选
1. **股价**: 10-50 元 (1 手 1,000-5,000 元)
2. **市值**: 50-500 亿 (避免小盘股风险)
3. **成交量**: 日均成交 > 1 亿 (保证流动性)
4. **行业**: 避开 ST、*ST、亏损股

#### 技术面筛选
1. **趋势**: 股价在 20 日均线上方
2. **波动**: 近 5 日振幅 < 15% (避免剧烈波动)
3. **指标**: MACD、RSI、均线同时发出买入信号

### 3.4 风险控制

#### 仓位控制
```
总仓位不超过 80%
单只股票不超过 50%
单日亏损不超过总资金 3%
```

#### 止损策略
- **硬性止损**: -5% 无条件卖出
- **技术止损**: 跌破关键支撑位
- **时间止损**: 买入后 5 日未盈利，考虑离场

#### 止盈策略
```
目标 1: +10% → 卖出 50%
目标 2: +15% → 卖出剩余
 trailing stop: 最高点回撤 5% 时卖出
```

## 四、A 股特殊注意事项

### 4.1 T+1 制度
- **当日买入无法卖出**: 需提前判断，避免冲动交易
- **最佳买入时间**: 14:30-14:50 (观察当日走势)
- **避免早盘追高**: 防止日内大幅回落

### 4.2 涨跌停限制
- **涨停板**: 可能次日高开
- **跌停板**: 可能继续下跌，优先止损
- **ST 股票**: 波动较小，不适合短线

### 4.3 交易时间
- **集合竞价**: 9:15-9:25 (不可交易，可挂单)
- **早盘**: 9:30-10:00 (波动最大，风险高)
- **午盘**: 13:00-13:30 (消化消息)
- **尾盘**: 14:30-15:00 (趋势形成，适合决策)

### 4.4 交易成本
```
佣金: 0.03% (最低 5 元)
印花税：0.05% (卖出时收)
过户费：0.001%

10,000 元买入成本:
- 佣金：5 元 (最低收费)
- 买入总成本：10,010 元

10,000 元卖出成本:
- 佣金：5 元
- 印花税：5 元
- 卖出总成本：10,010 元
```

## 六、预期收益与风险

### 6.1 收益测算

```
假设条件:
- 月交易次数：4 次
- 胜率：60%
- 平均每笔盈利：10%
- 平均每笔亏损：-5%

月度收益:
- 盈利交易：2.4 次 × 10% = 24%
- 亏损交易：1.6 次 × -5% = -8%
- 净收益：16% × 10,000 元 = 1,600 元

年度收益 (复利):
(1 + 16%)^12 - 1 = 5.5 倍 (理论值，实际需扣除损耗)
```

### 6.2 风险控制

```
月度最大亏损 (胜率 40%):
- 盈利交易：1.6 次 × 10% = 16%
- 亏损交易：2.4 次 × -5% = -12%
- 净收益：4% × 10,000 元 = 400 元

极端情况 (连续 5 次止损):
- 亏损：5 × 5% × 10,000 元 = 2,500 元
- 剩余资金：7,500 元
- 恢复难度：+33% 回本
```

### 6.3 心理建设

1. **严格执行纪律**: 不犹豫止损，不贪婪止盈
2. **接受亏损**: 亏损是交易的一部分
3. **保持耐心**: 等待最佳机会
4. **持续学习**: 总结每笔交易经验
